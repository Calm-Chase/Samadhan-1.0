# Samadhan-1.0
Chatbot Using Large Language Model 

To build your env 

For Windows 
1. python -m venv env 
2. env\Scripts\activate
3. pip install -r requrements.txt


For Linux 

1. python3 -m venv env 
2. source env\bin\activate
3. pip install -r requrements.txt

For Docker 
docker compose up --build
